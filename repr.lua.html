M.repr (value)

Generate a readable string out of any lua-value. It is almost serialisation ;)

Examples: 

return repr(nil)
"nil"
return repr(1)
"1"
return repr "test string"
"\"test string\""
return repr {1, 2, b=7, 3, a=6,}
"{1, 2, 3, a=6, b=7}"
return repr(function () end)
"function: 0x10260d0"
